# 🍽️ Deju's Restaurant Page

A modern restaurant website built with vanilla JavaScript and Webpack.

## 🚀 Live Demo

[https://Fortress-io.github.io/AuraX-Summer-2026]

## 📌 Features

- Single Page Application behavior
- Dynamic page rendering without reloads
- Home, Menu, and Contact pages
- Responsive menu card layout
- CSS Grid-based menu design
- Imported images using Webpack assets
- Modular JavaScript architecture

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Webpack
- npm

## 📚 What I Learned

- Setting up and configuring Webpack
- Using ES Modules
- Managing project dependencies
- Creating reusable DOM components
- Separating application responsibilities
- Building dynamic interfaces with JavaScript
