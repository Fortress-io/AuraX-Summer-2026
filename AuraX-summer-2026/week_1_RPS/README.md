# Rock Paper Scissors 

A simple console-based Rock Paper Scissors game built with JavaScript as part of The Odin Project Foundations and the AuraX Summer Program.

## Features

* Play against the computer.
* Computer makes random choices.
* Case-insensitive user input.
* Keeps track of scores.
* Plays 5 rounds automatically.
* Displays the final winner.

## Technologies Used

* HTML5
* JavaScript (ES6)
* Browser Developer Console

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/Fortress-io/AuraX-Summer-2026
```

2. Navigate to the project folder:

```bash
cd AuraX-Summer-2026/week_1_RPS
```

3. Open `RPS.html` in your browser.

4. Open the browser Developer Console:

* Windows/Linux: `F12` or `Ctrl + Shift + J`
* Mac: `Cmd + Option + J`

5. The game starts automatically and prompts you to enter:

```text
rock
paper
scissors
```

## Game Rules

* Rock beats Scissors.
* Scissors beats Paper.
* Paper beats Rock.
* The game consists of 5 rounds.
* The player with the highest score wins.
