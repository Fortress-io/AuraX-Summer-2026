# Calculator

A simple calculator web application built using HTML, CSS, and JavaScript.  
This project allows users to perform basic arithmetic operations through an interactive user interface.

## Project Overview

This calculator was created as a practice project to understand JavaScript fundamentals, especially:

- DOM manipulation
- Event listeners
- Functions
- Variables and data handling
- User interaction with web elements

The calculator provides a clean interface where users can enter numbers, select operations, and calculate results.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear display functionality
- Decimal number support
- Interactive buttons
- Error handling for division by zero
- Responsive calculator layout

## Technologies Used

### HTML

Used to create the structure of the calculator, including:

- Display screen
- Number buttons
- Operator buttons
- Control buttons

### CSS

Used for styling and layout:

- CSS Grid for calculator arrangement
- Button positioning
- Colors and borders
- Display formatting

### JavaScript

Used to add functionality:

- Selecting HTML elements using DOM
- Handling button clicks
- Storing numbers and operators
- Performing calculations
- Updating the display dynamically
