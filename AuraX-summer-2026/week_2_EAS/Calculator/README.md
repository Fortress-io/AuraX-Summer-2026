# JavaScript Calculator

A fully interactive calculator built with **HTML, CSS, and JavaScript** as part of my web development learning journey.

This project focuses on strengthening core JavaScript concepts such as DOM manipulation, event handling, state management, and problem-solving by recreating the behavior of a real calculator.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Division-by-zero error handling
- 🔢 Multi-digit number input
- 🔘 Decimal number support
- 🧹 Clear (`C`) button
- 🧠 Chained calculations (e.g. `5 + 3 + 2 =`)
- 🔄 Automatically starts a new calculation after displaying a result
- Keyboard support

- Number keys (`0–9`)
- Operators (`+`, `-`, `*`, `/`)
- Decimal (`.`)
- `Enter` or `=` to calculate
- `Escape` to clear
- `Backspace` to delete the last digit

## Built With

- **HTML5** — Structure
- **CSS3** — Layout and styling using CSS Grid
- **JavaScript (ES6)** — Calculator logic and interactivity

## How It Works

1. The user enters numbers by clicking buttons or using the keyboard.
2. The first number is stored.
3. An operator is selected.
4. The second number is entered.
5. Pressing **=** or **Enter** performs the calculation.
6. The result is displayed.
7. The calculator is ready for either a new calculation or another chained operation.
