# Etch-a-Sketch Project

A simple browser-based drawing app built with HTML, CSS, and JavaScript.  
Users can draw by hovering over a grid of squares, similar to a digital Etch-a-Sketch.

## Features

- Dynamic 16×16 grid generation using JavaScript
- Hover-to-draw functionality (black color)
- Reset button to clear the drawing
- Responsive grid layout using Flexbox
- Clean separation of HTML, CSS, and JavaScript

## How It Works

- JavaScript generates a grid of div elements dynamically
- Each square listens for a `mouseover` event
- When hovered, the square changes color to simulate drawing
- Reset button clears all colors without rebuilding the grid
